3. Data Cleanup Applied

  • 1
    ILT dual-authorization merge — 11 companies held both P (Private) and G (Governmental) authorizations as separate rows. Merged into single record with auth_numbers array. No data lost.
  • 2
    Cross-source priority ordering — ILT OA (score 30), DCRO (50), and ILT STS (65) all map to aviation_authority. 27 companies appeared in both DCRO + ILT OA. Processed DCRO first so higher-score record wins.
  • 3
    "LUC" keyword false positive — Dutch words "luchtvaart" (aviation) and "luchtfotografie" (aerial photography) contain substring "luc". Fixed with word-boundary matching to only detect "LUC" as a standalone certification keyword.
  • 4
    Drones.nl domain collision — All 52 Drones.nl entries had website = drones.nl/bedrijven/... which would merge all into one company. Set normalized_domain = null for this source.
  • 5
    Name mismatch across sources — Same company with different names (e.g., "Falcker Drone Robotics" vs "Falcker Innovations" vs "Falcker") resolved by domain-based dedup in grouped API (all share falcker.com).
  • 6
    58 records with no website/LinkedIn — 55 ILT OA + 3 ILT STS. These are not eliminated — they remain eligible for all tiers and the top 30. Missing contact data is an enrichment gap, not a disqualification. Queued for website/LinkedIn enrichment via Apollo, LinkedIn search, or manual lookup.
Enrichment approach: No company is excluded from ranking due to missing website or LinkedIn. Blank fields are enrichment opportunities, not elimination criteria. After the top 30 is finalized, we run enrichment (Apollo / LinkedIn lookup / manual search) to fill gaps before BD outreach.
