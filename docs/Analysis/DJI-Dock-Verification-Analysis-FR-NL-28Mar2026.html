Step 1: Serper Website Verification (Primary)

  • Query: site:domain "DJI Dock" — exact phrase match ensures DJI-specific results
  • Deduplication: Records grouped by domain before verification — one API call per unique domain, results fanned out to all matching records
  • Variant detection: Regexes require "DJI" context within 30 characters of "Dock 2/3" to prevent false positives from shipping docks, loading docks, etc.
  • Relevance classification:
    • direct — URL contains /product/, /shop/, /dji-dock, or snippet mentions price/buy/order
    • indirect — Snippet mentions "vs", "alternative", "comparison"
    • mention_only — Generic blog or news mention
  • Evidence priority: Product page URL > URL with "dock" in path > first organic result
  • Cost: 1 Serper credit per unique domain (0 for duplicates)

Step 2: Puppeteer LinkedIn Deep Scan (Selective)

  • Trigger: Only runs on companies where Step 1 found 0 hits AND company came from a high-value source (DJI Dealer, Google Search, ChatGPT, Comet)
  • Method: Headless Chrome with LinkedIn cookie auth → navigates to company /posts/ page → scrolls → scrapes post text
  • Keywords: Strictly /dji\s*dock/i, /dji\s*dock\s*[123]/i — no generic "dock", no BVLOS, no drone-in-a-box
  • Captures: Post URL (clickable evidence), post date (for freshness analysis), full snippet text, dock model variants
  • Limitations: Cookie expiry (~24h), rate limiting (~40 companies/session), CAPTCHA risk
  • Circuit breaker: 3 consecutive LOGIN_WALL/CAPTCHA → stop batch, notify user
  • Cost: 0 API credits (uses browser session)

What the Agent Cannot Do (Human Judgment Still Required)

  • Qualify partner fit: A confirmed DJI Dock reseller in Japan may not be relevant to FlytBase's immediate market entry — the team decides priority.
  • Interpret indirect mentions: A company blogging about DJI Dock comparison vs Percepto is a signal of interest but needs context — are they evaluating, reviewing, or selling?
  • Validate ChatGPT/Comet pre-loaded data: These sources may have dated information; the agent flags 0-hit records for manual follow-up.
  • LinkedIn cookie management: Puppeteer requires periodic LinkedIn session refresh (~every 24h).
