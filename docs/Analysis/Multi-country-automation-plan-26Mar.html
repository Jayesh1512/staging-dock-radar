3 Google Search Campaign Automated batch

NEW: Batch Campaign replaces manual per-country runs

  • Takes a list of countries from config
  • Runs Google Search Crawler for each country with keyword “DJI Dock”, 7 pages
  • Stores ALL results in a google_search_runs staging table (not directly in source_candidates)
  • Campaign dashboard shows: country, entities found, scored, ready to import
  • User reviews per-country results, clicks “Import to Pipeline” per country

Key design decision: Staging table captures everything. Import step applies filters (noise, foreign TLD, media, junk slugs). This keeps the pipeline clean while preserving raw data.
Human touch: Review campaign results, approve import per country (~5 min per country).
