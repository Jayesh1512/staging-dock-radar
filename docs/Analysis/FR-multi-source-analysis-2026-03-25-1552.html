Methodology
1. SIRENE Registry (365 records) — French govt business registry. Filtered by drone-related NAF codes (71.12B, 71.20B, 30.30Z, etc.) + keyword matching ("drone", "UAV", "RPAS"). Two-track waterfall: Track A (NAF + keywords, high confidence) + Track B (keywords only, medium). No website/LinkedIn available from registry.
2. DJI Reseller List (12 records) — Official DJI Enterprise Dealers extracted from dji.com/where-to-buy. Includes website, LinkedIn, city. Fixed score 80, entity_type: reseller.
3. Comet Intelligence (18 records) — Industry intelligence data. Includes DJI Dock 3 authorization status (2 authorized: Flying Eye, Abot). Score 80-85.
4. Google Search Crawler (14 records) — Serper.dev API, query "DJI Dock" France, 5 pages. Entities extracted from Google results, scored 0-100 by tiered keyword matching (Tier 1: dji dock/dock 2/dock 3, Tier 2: bvlos/sora, Tier 3: industry verticals). Entity type via v5 two-tier classification.
5. Cross-Source Merge — Companies matched by normalized_domain (primary) and normalized_name (secondary). Source count = number of independent sources confirming the company. Ranking: source_count×30 + best_score + contact_surface_bonus.
Entity Classification (v5) — Two-tier operator signals. Primary (service-delivery language: intégrateur, prestation, consulting) + Secondary (product-feature language: autonome, sécurité, mission). Primary + Secondary → DSP/SI. Primary only + Reseller → Reseller. Secondary only + no Reseller → DSP/SI. Domain patterns (.store, shop., DJI slug) override content.
