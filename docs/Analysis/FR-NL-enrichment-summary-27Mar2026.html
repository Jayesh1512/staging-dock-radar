Technical Notes

  • Name cleaning: Legal suffixes (SAS, SARL, EURL, B.V., NV, LLC, etc.) stripped before Apollo lookup. Parenthetical trade names tried as variants.
  • Apollo endpoint fix: Previous runs used GET /organizations/enrich?organization_name=X which requires a domain parameter (returns 422 without it). Switched to POST /mixed_companies/search which accepts company names — this is why previous runs returned 0 results.
  • Track B optimization: Apollo domain-enrich tried first (free), Serper only used as fallback when Apollo didn't return LinkedIn.
  • Track C quality control: 20/20 Serper domain lookups returned results, but 15 were false positives (directory sites, unrelated pages). These were automatically reverted, keeping only 5 verified domains.
  • Rate limiting: 300ms between Apollo calls (~200/min), 500ms between Serper calls. No rate limit errors encountered.
  • DB updates: Each successful enrichment sets enriched_at timestamp and updates dock_qa_status to 'enriched' when website is found.
  • API route created: POST /api/registry/enrich-batch — reusable for future country enrichment. Accepts {countryCodes, dryRun}.
